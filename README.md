# 📝 Case Diary Management System

## 📌 Project Overview

The Case Diary Management System is a web-based application designed for police departments and investigation agencies.
The main objective of this system is to digitally manage, track, and maintain case records efficiently and securely.

This system helps reduce manual paperwork and improves the overall efficiency of case handling.

---

## 🚀 Features

* 👤 User Authentication (Login / Signup)
* 📂 Case Record Management
* ➕ Add New Case
* ✏️ Update Case Details
* ❌ Delete Case Records
* 🔍 Search & Filter Cases
* 📊 Case Status Tracking
* 🔐 Secure Data Handling

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

git clone https://github.com/Swagat-190/Case-Diary-Management.git

---

### 2️⃣ Frontend Setup

cd frontend
npm install
npm start

---

### 3️⃣ Backend Setup

* Open the backend folder in IntelliJ IDEA or Eclipse
* Run the Spring Boot application

---

### 4️⃣ Database Setup

* Open MySQL
* Create a database (e.g., `case_diary`)
* Import the SQL file (if available)

---

## ▶️ How to Run the Application

1. Start the backend server
2. Start the frontend
3. Open your browser and go to:
   http://localhost:3000
